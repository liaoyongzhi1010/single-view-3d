---

### 🌟 Highlights

- 💡 提出 SSR 框架：结合神经隐式形状建模与辐射场（Neural Implicit Shape + Radiance Field）。
- 🔁 使用两阶段训练（3D supervision + 2D rendering supervision），有效解决形状-外观歧义问题。
- 📈 在 3D-FRONT 和 Pix3D 数据集上重建精度分别提升 27.7% 与 11.6%。
- 🧩 支持多视角渲染、编辑、与场景理解等多种下游任务。
